# Autism Spectrum Disorder Detection Using Deep Learning Models

## Overview

Autism Spectrum Disorder (ASD) is a neurodevelopmental condition that affects communication, social interaction, and behavior. Early detection can support timely intervention and improve developmental outcomes.

This project presents a Django-based web application that detects Autism Spectrum Disorder (ASD) from facial images using deep learning. The study evaluates the performance of multiple transfer learning models and integrates the trained model into a web application for real-time prediction.

> **Note:** This application is intended as an assistive screening tool for educational and research purposes. It should not be used as a substitute for professional medical diagnosis.

## Deep Learning Models Evaluated

- ResNet-18
- EfficientNet-B1
- EfficientNet-B2

Among the evaluated models, **ResNet-18 achieved the highest classification accuracy**, while the deployed application integrates the trained deep learning model for prediction.

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Autism-Spectrum-Disorder-Detection-Using-Deep-Learning-Models.git
```

2. Navigate to the project directory

```bash
cd Autism-Spectrum-Disorder-Detection-Using-Deep-Learning-Models
```

3. Install the required dependencies

```bash
pip install -r requirements.txt
```

4. Start the Django development server

```bash
python manage.py runserver
```

5. Open your browser and visit

```text
http://127.0.0.1:8000/
```
